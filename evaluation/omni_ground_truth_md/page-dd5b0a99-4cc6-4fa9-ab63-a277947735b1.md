# # 二、消费者的地理标志农产品认知分析

# ## (一) 消费者对地理标志农产品信息的关注程度

在调查中，77.2％的被调查者选择“关注有关地理标志农产品的新闻报道信息”，而22.8％的被调查者选择了“不关注”，说明大部分的消费者对农产品的质量安全问题是普遍关注的。就消费者对地理标志农产品信息获取渠道而言，有21人选择“凭个人经验或亲友获得信息”，占到被调查者的10.88％其次，选择销售商或者商场获取信息的比例占27.98%，选择“书报、展览会等获取地理标志农产品信息”和“网络、广播电视提供地理标志农产品信息”也占到了一定的比例，分别为24.35％和10.88%，而选择“由政府提供的地理标志农产品信息”所占的比例为25.91％。以上调查结果显示，消费者选择销货商或商场提供的信息比例较高，这可能与企业的轰炸式广告有关，然而，选择政府部门提供信息仅占总样本的1/4政府公信力有待加强。

# ## (二) 消费者对当前我国食品安全状况的评价

调查结果显示，大部分的消费者对当前市场上销售的农产品质量表示非常担忧。在回答“如何看待当前我国的食品安全状况？”时，选择“良好”的仅有7人，占3.63％。选择“一般”的占16.06%，选择“较差”和“非常差”的分别占30.57％和49.74％。这与近些年频发的食品安全问题有关，农产品质量的负面新闻越多，越容易

造成消费者对农产品质量安全状况信心不足。

# ## (三) 消费者对地理标志农产品附加值信息的关注程度

在农产品附加值信息中，消费者选择最关注的信息依次有: 营养价值(56%),环境保护 ( 45% ) 历史文化体验感( 37% ) 。在农产品附加值信息中，产品的营养价值是消费者最关注的附加值信息。其原因可能是消费者消费层次和消费观念发生巨大的转变， 农产品的营养效果会加强消费者的认知和购买意愿 $ {}^{\left\lbrack {11}\right\rbrack } $ 。消费者关注程度最低的是历史文化体验感，这与传统文化的传承危机有关。历史沉淀是地理标志农产品的内在竞争力，具有极强的品牌价值，对提高消费者的认同感具有较大的促进作用。

# ## (四) 消费者对地理标志农产品质量安全 的关注程度

地理标志认证是当前我国安全农产品认证的基本形式之一。通过调查可以发现，对其标识非常了解的比例在55.4%，不关注地理标志农产品标识的比例为44.6％。这说明近一半的消费者对地理标志农产品的标识不了解。在消费者被问到是否知道地理标志农产品要经过严格的申请、检查等程序时，有77.2％的消费者选择“知道”。
