# 百里山水入画来

“百里山水画廊”这个充满诗情画意的名字，已在不经意间走进了北京市民的生活，成为市民休闲旅游所向往的乐园。

千家店镇位于延庆东部深山区，空气清新、山峦重叠、水景众多，林木绿化率达到 82%，拥有硅化木国家地质公园、滴水壶，乌龙峡谷等众多自然景观和朝阳寺等历史文化遗迹，形成了秀水湾、长寿岭、古家窑等特色民俗村。

2008年底，根据本地自热条件、产业基础、历史文化等条件。经过专家多次考察论证，千家店

镇决定对域内店多自热资谅进行合理有效地规划，针对千家店沟域纵深长、面积大，内容多、功能全的特点，将整条沟域确定为”百里山水画廊。2009年，共投资 5000多万元.按照“点、线、面”梯次推进，系统建设十二个节点。一期完成了干沟，秀水湾、镇区。古家山寨、滴水壶、乌龙峡谷六个节点的建设。道路两侧建设现量台16个，水文景观5处，改造沿线 14个村庄外立面11万平方米，立节点景观标志2处.以向日葵为主，调整农业种植结构6800亩，大尺度地营造大地景观。

百里山水画廊的建设，使传统农业向深度一二三产相结合的多功能农业转变。在千家店镇起稻的农田里，1.4万亩杏树成长起来，每当杏花开时，全镇飘香；杏树下万亩黄芩，成片的紫色花海让人沉醉；当七八月份千亩葵海盛开时.这里又成了摄影爱好者的天堂。这些农田不但具有经济价值，还具有观赏价值，成了名副其实的生态观光农业园区，也是风景独特的旅游资源，前来观光的游客络绎不绝。

市民来了，城里的资金和智力优势也随之而来。投资1000万元修建的”艺术家交流中心”已投入使用，户外写生基地正在形成。日前，千家店已与北京20所高校签订写生协议，来自中央美术学院、中央工艺美院附中两所学校的近400名师生首批入驻百里山水画廊，开始户外写生，其它18所高校也将陆续入驻。

# 论乡村旅游开发中经济效益与生态效益的博弈（上）

# 白偶风  任荣

乡村旅游是指利用田园景观，农耕活动以及农耕文化等乡村特有的物质和文化资源，吸引都市人群回归自然和乡土，享受返璞归真的农耕活和文化趣旨的旅游活动。乡村旅游是农村实现经济结构转型的重大举措。合理开发乡村旅游，不仅有利于农民增收，还有利于农村生态文明建设。然而，在巨大的经济利益刺激下，当前乡村旅游呈现出一窝蜂的态势。部分乡村旅游有名无实，为了追求短期利益，无视生态环境的日益恶化，为了牟取少部分的私利，而置公共利益于不顾。这完全抛弃了乡村旅游以维护清洁的自然环境和宝贵的乡土资源的前提，也背离了振兴农村经济和帮助农民致富的宗旨。乡村旅游开发的矛盾根源，在于开发过程中经济效益与生态效格的失衡，各利相关者之间利益分配不合理和责任不明。乡村旅游开发，实质是一局多方参与的博弈。坚持可持续发展道路，是实现各方共赢共存局面的不二选择。

# 商机中诞生的乡村旅游

工业革命以后，随着社会分工的逐步细化，城乡二元化趋势的强，生活在市中的人们被水泥森林围困，不仅怀念都市外的一抹青山秀水，更向往田园生活的安宁和惬意。这种心理需求最终被一些具有商业头脑的人理解为一种消费者的诉求，而这种诉求孕育着极大的商机，因此正是在这种消费者诉求基础上诞生了乡村旅游。农业本身是生产利润低于社会平均利润的产业，农业的这一性质决定在市场经济体制下，传统农业一且有机会就会发生转型，农业用地将不只是践行农业生产这一单一功能。乡村旅游正是将生产利润较低的农业与利润较高的旅游业相结合.实第一产业向第三产业的转型。

# “无利不起早“的乡村旅游开发者

从消费者的角度讲，乡村旅游的产生使自己的消费意愿得到满足。而站在开发者的角度，更多地是关心这个巨大的商机背后将会带来么丰厚的经济收益。当前，我国乡村旅游正处于蓬勃发展阶段，已创造出巨大的经济效益。每年的”五一”“十一”和春节三个旅游“黄金周”，全国城市居民出游中选择乡村旅游的约占70%，每个“黄金周”可形成约6000万人次规模的乡村施游市场。由国家旅游局在全国倡导的创建全国农业旅游示范点已达到359家，遍布内地31个省市区，覆盖了农、林、牧、副、渔及种植业、养殖业、加工业等农业的各种业态。据测算，目前全国乡村旅游景区（点）每年接待游客超过3亿人次，旅游收入超过400亿元人民币。

面对如此庞大的国内乡村旅游市场.手中拥有资源或持有资金的开发者们再也按接不住了，纷纷将手中的资源和资金投向乡村旅游市场，期望能

着大好势头睡得贫满滋。正所谓”无利不起早”，开发者总是希望以最短的时间最小的投入获取晶大的回报，这就导致了经营的短整性和利益的私人化。完全由市场主导的乡村旅游，由于开发者的趋利心理，容易造成开发的不合理和竞争的无序。而由此产生的堂皇的宏观经济数据往往给人造成象，不能反映产业的可持续前景。实际上，宏观经济数据的持续走红暴露了开发者竞相车利的心态掩盖了乡村旅游开发中的种种隐思。

# 自然生态和文化生态的退化

公地悲剧。“公地悲剧”是指单纯地追求最大化利益的个体在没有相应制度约束下，其短期行为对公共利券造成的损失。乡村旅游资源的产权属性是积有产权，公共产权的结合。房产及家庭财富是积有财产，而当地整体自然环境及人文环境、乡村旅御品牌等期是相关主体共同拥有的财富。由于不能排他地使用，乡村中的每个成员以及外来投资者都可以利用这些公共财富。在缺乏相应的机制管束下，农户之间，投资者之间展开了对公共资源的无序争夺和使用，使得原本基础设施条件就相当薄要的乡村资过度开发，环境发生退化：足血导镇公理悲用广：在乡村成游地发展中“公地乘国主要委现安环货表化资税过度使用、退化和潜在毁天等问题，严重损害了旅地的形象，动摇了乡村旅游的根基。

环境值。任何一个生态表统的自净能力都是有限的，它的上限被称为“环境阈值”。当环境中污染物的排放超过其自净能力，也就是超过环境的承载阈值”时，那么，人类的经济活动将受到环境的限制，甚至想到环模的报复，经济增长无法持续，人类福祉也会下降。而且值得注意的基，环境鱼值并不是固定的静态的，它随升环竞污染优配的恶化而不断降低。这意味着环境对经济增长的负面影响时间随着环境状况的恶化而提前。乡村环境的环境黄值“给乡村旅游制定了一个理性的钧史，那就量游客的接特量小能超出环模的承载力。一味地追求经济利益而忽视环境承转力.官目地开发和扩大经营，已经使得不少多村旅游不境递渐恶化吸引力递渐弱化

# 刘斌堡乡涌现一批“文凭农民”

2010年12月15日.延庆县刘第堡乡大观头村“首届旅游服务中专班毕业典礼”在村委会举行。这个日子标志着，全村56名农村学员经过三年的学习，正式从中央农业广播电视学校毕业，取得国家承认学历的中专毕业证书。这些农民不离多土，不离农事，在广阔田园边务农边读书，破天荒地成为“文凭农民”。范长瑜

# 木林镇三项措施繁荣乡土文化

自去年以来，顺义区木林的积极采取三项措施，促进群众文化事业发展；一是进一步完养群余文化基础设拖，如建立图书室改造致字影厅、完善体育健身器材等；二是不断创新文化活动形式和内容，把木林销的文化联碧充分控期出来；三是拍性专业文艺团队，让更多群众参与其中，真正丰富了村民们的精神文化生活。郑晚川

# 沈家营镇中专教育进了村

去年12月10日上午.延庆且沈家营销前吕庄村大队部举办了种植和文化中专班开办典礼，65名种田人足不出村就可以像城里人一样，接变系统的中专学历教育。学制为三年，实行学分制，完成学分经考核合格后，可获得国家承认的中专毕业证书。刘源源

# 北郎中村沼气站跨人大型行列

近日，顺义区赵全营镇北郎中村环能工程沼气站，以每天生产2000支方米的成绩跨人了北京市大型沼气站的行列。该气站从2002年开始建设，目前已经完成三期工程的建设，共投资1500万元。翁福文  胡悦

# 首钢冷轧厂实现水资源循环利用

近日，夏文区李舒镇百钢冷乱理假有限公司荣获“”顺义区节水型先进单位”称号。据了解。“十一五”以来，首钢冷轧厂坚持*开源与节流井重、节流优先、治污为本、科学开源，综合利用”的原则，以科技促节水，使水资源利用效率显著提高.企业年用水指标从280万立方米减少到22万立方米，年使用再生水，雨洪水75.60万吨。王娜  姚治国
