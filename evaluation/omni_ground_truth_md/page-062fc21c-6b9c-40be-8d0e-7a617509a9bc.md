# ### 1.1.9【因式分解双换元类型】

# # $\blacktriangle$适用题型：

双变量条件等式能因式分解的时候可以考虑双换元。整理成更容易利用不等式的结构；

# # $\blacktriangle$方法原理：

比如：$\left\{  \begin{array}{l} {a}^{2} - {b}^{2} = 1 \Rightarrow  \left( {a + b}\right) \left( {a - b}\right)  = 1 \Rightarrow  \left\{  \begin{array}{l} a + b = m \\  a - b = \frac{1}{m} \end{array}\right. \\   \Rightarrow  \left\{  \begin{array}{l} a = \frac{1}{2}\left( {m + \frac{1}{m}}\right) \\  b = \frac{1}{2}\left( {m - \frac{1}{m}}\right)  \end{array}\right. \\   \Rightarrow  {2a} + b = \left( {m + \frac{1}{m}}\right)  + \frac{1}{2}\left( {m - \frac{1}{m}}\right)  = \frac{3m}{2} + \frac{1}{2m} \end{array}\right. $

关键在于观察条件等式能否因式分解,除了常见的初中所学的因式分解,还有一些可以配凑分解,比如 $\left\{ \begin{array} { l l } { a b + a + b = 3 \Rightarrow a b + a + b = a { \left( \ b + 1 \right) } + b + 1 - 1 } \\ { = { \left( a + 1 \right) } { \left( b + 1 \right) } - 1 = 3 \Rightarrow { \left( \ a + 1 \right) } { \left( \ b + 1 \right) } = 4 } \end{array} \right.$

例题1.（辽宁·高二期末）若实数${ a , b }$满足$4 a ^ { 2 } - b ^ { 2 } = 4$，则$5 a ^ { 2 } + 2 a b$的最小值为____.

例题2．（全国·高一单元测试）已知正实数$a , ~ b$满足$a b + a + b = 3$，则$2 a + b$的最小值为____.

例题3．（山东德州·高二期末）若$a > 2 , b > - 1$，且满足$a b + a - 2 b = 6$，则$\frac { 1 } { a - 2 } + \frac { 9 } { b + 1 }$的最小值为____.

练习1．（全国·高一单元测试）已知正实数$x , \ y$满足：$x ^ { 2 } + x y + \frac { 2 x } { y } = 2$，则$3 x + 2 y + { \frac { 2 } { y } }$的最小值为____.
