# ### 2.3.13【数形结合法之圆】

# #### ▲适用题型：

常见于含有 $\sqrt { r ^ { 2 } - \left( x - a \right) ^ { 2 } } + b$ 结构的题型；

# #### ▲方法原理：

令 $ y = { \sqrt { r ^ { 2 } - \left( x - a \right) ^ { 2 } } } \Rightarrow y = { \sqrt { r ^ { 2 } - \left( x - a \right) ^ { 2 } } } \geq 0 \Rightarrow { \left\{ \begin{array} { l l } { y \geq 0 } \\ { \left( x - a \right) ^ { 2 } + y ^ { 2 } = r ^ { 2 } } \end{array} \right. }$即得到半圆，

1. 若是分式类型，常将问题转换为斜率问题，比如$f \left( x \right) = { \frac { \sqrt { 4 - { \left( x - 3 \right) } ^ { 2 } } - 2 } { x + 2 } }$，

(1) 令 $ y = \sqrt{4 - {\left( x - 3\right) }^{2}} \Rightarrow  y = \sqrt{4 - {\left( x - 3\right) }^{2}} \geq  0 \Rightarrow  \left\{  \begin{array}{l} y \geq  0 \\  {\left( x - 3\right) }^{2} + {y}^{2} = 4 \end{array}\right. $

(2)$  \Rightarrow  \left\{  \begin{array}{l} f\left( x\right)  = \frac{\sqrt{4 - {\left( x - 3\right) }^{2}} - 2}{x + 2} = \frac{y - 2}{x - \left( {-2}\right) } \\  \text{ 令 }A\left( {x,y}\right) ,\text{ 且 }A\text{ 在半圆上,}~B\left( {-2,2}\right) \\  {k}_{AB} = \frac{{y}_{A} - {y}_{B}}{{x}_{A} - {x}_{B}} \end{array}\right\} \Rightarrow  f\left( x\right)  = {k}_{AB} $

最后作图分析求出AB斜率范围即可；

2. 若是整式类型,常将问题转换两函数图像问题，比如 $f \left( x \right) = 3 x - 1 - \sqrt { - x ^ { 2 } + 4 x - 3 }$

(1) $\left\{ \begin{array} { l l } { f ( x ) = 3 x - 1 - \sqrt { - x ^ { 2 } + 4 x - 3 } = 3 x - 1 - \sqrt { 1 - \left( x - 2 \right) ^ { 2 } } , } \\ {令t = f ( x ) \Rightarrow \sqrt { 1 - \left( x - 2 \right) ^ { 2 } } = 3 x - 1 - t } \end{array} \right.$

(2) $\begin{cases}
\text{令}y_1=\sqrt{1-\left(x-2\right)^2}\geq0\Rightarrow
\begin{cases}
y\geq0 \\
\left(x-2\right)^2+y^2=1 & 
\end{cases} \\
\text{令}y_2=3x-1-t,\text{则}y_1=y_2 & 
\end{cases}$ $\Rightarrow { y } _ { 1 }$ 和 $y _ { 2 }$ 有交点时，求纵截距 $ { - t }$ 范围

最后作图分析即可求出$-t$范围，也就求出 $ { t }$范围；
