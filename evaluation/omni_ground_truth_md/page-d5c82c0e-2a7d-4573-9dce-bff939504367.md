The following proposition describes the close relationship between the Nijenhuis operator and the Rota-Baxter operator. Similar results hold for Hom-associative conformal algebras (see prop 6.2 of [4]). Let us observe, if these results hold for Hom-Leibniz conformal algebras or not.

Proposition 2.10. Let $ {\mathcal{N}}_{h} : \mathcal{L} \rightarrow \mathcal{L} $ be a $ \mathbb{C}\left\lbrack \partial \right\rbrack $ -module homomorphism over a Hom-Leibniz conformal algebra $ \mathcal{L} $ . Then

(1) $\mathcal { N } _ { h }$ is a Nijenhuis operator if and only if $\mathcal { N } _ { h }$ is a Rota-Baxter-operator of weight 0, given that $\mathcal { N } _ { h } ^ { 2 } = 0$ .

(2) $\mathcal { N } _ { h }$ is a Nijenhuis operator if and only if $\mathcal { N } _ { h }$ is a Rota-Baxter-operator of weight ${ - 1 }$,  given that $\mathcal { N } _ { h } ^ { 2 } = \mathcal { N } _ { h }$ .

(3) $\mathcal { N } _ { h }$ is a Nijenhuis operator if and only if $\mathcal { N } _ { h }$ is a modified RB-operator of weight $\mp1$, given that $\mathcal { N } _ { h } ^ { 2 } = \pm I d$.

(4) $\mathcal { N } _ { h }$ is a Nijenhuis operator if and only if $\mathcal N _ { h } \pm I d$ isa $R B$ -operator of weight $\mp2$, given that $\mathcal { N } _ { h } ^ { 2 } = \mathit{I d}$.
