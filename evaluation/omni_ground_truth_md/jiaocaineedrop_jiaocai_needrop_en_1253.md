# 误区讲解

教室的面积一定，方砖的面积和所用砖的块数成反比例。

用边长为 15 cm的方砖给教室铺地，需要2000块，如果改用边长为 25 cm的方砖铺地，需要多少块？（用比例知识解答）

# 错误解答：

解：设需要 x 块方砖。

$$
$$
\begin{array}{c}
2 5 x \!=\! 1 5 \! \times\! 2 0 0 0
\\x \!=\! 1 2 0 0
\end{array}
$$
$$

错因分析：这题用比例知识列的方程是错误的，本题成反比例的两个量不是方砖的边长和方砖的块数，而是方砖的面积和方砖的块数。

# 正确解答：

解：设需要 x 块方砖。

$$
$$
\begin{array}{c}
2 5^{2} \ x \!=\! 1 5^{2} \! \times\! 2 0 0 0
\\x \!=\! 7 2 0
\end{array}$$
$$
