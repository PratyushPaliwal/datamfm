# ### 2.3.17【柯西不等式法】

# ▲适用题型:

常用于根式中有和为定值时，比如求 $f ( x ) = { \sqrt { m { \bigl ( } x + p { \bigr ) } } } + { \sqrt { n { \bigl ( } - x + q { \bigr ) } } }$ 最大值；

# ▲方法原理:

1. 二维柯西不等式：

设 $ a ,  b ,  c ,  d$ 均为实数，有 $( a ^ { 2 } + b ^ { 2 } ) ( c ^ { 2 } + d ^ { 2 } ) \geq ( a c + b d ) ^ { 2 }$ 当且仅当 ${ \frac { a } { c } } = { \frac { b } { d } }$ 时等号成立；

向量法证明: $\begin{cases}
\vec{m} = (a, b), \vec{n} = (c, d), |\cos \theta| \leq 1 \\
\Rightarrow |\vec{m} \cdot \vec{n}| = \|\vec{m}\| \vec{n} |\cos \theta| \leq|\vec{m}| |\vec{n}| \\
\Rightarrow |ac + bd| \leq \sqrt{a^2 + b^2} \sqrt{c^2 + d^2} \\
\Rightarrow (a^2 + b^2)(c^2 + d^2) \geq (ac + bd)^2
\end{cases}$；取等时向量共线，即 ${ \frac { a } { c } } = { \frac { b } { d } }$；

解答题用到柯西不等式，即可用此证明；

2. $\sf n$维柯西不等式：

$$
$$( a _ { 1 } ^ { 2 } + { a _ { 2 } } ^ { 2 } + { a _ { 3 } } ^ { 2 } + . . . . . . + a _ { n } ^ { 2 } ) ( b _ { 1 } ^ { 2 } + b _ { 2 } ^ { 2 } + b _ { 3 } ^ { 2 } + . . . . . . + b _ { n } ^ { 2 } ) \geq ( a _ { 1 } b _ { 1 } + a _ { 2 } b _ { 2 } + a _ { 3 } b _ { 3 } + . . . . . . + a _ { n } b _ { n } ) ^ { 2 }\text{ 字母均为}R,$$
$$

当且仅当 ${ \frac { a _ { 1 } } { b _ { 1 } } } = { \frac { a _ { 2 } } { b _ { 2 } } } = { \frac { a _ { 3 } } { b _ { 3 } } } = \ldots \ldots= { \frac { a _ { n } } { b _ { n } } }$ 时等号成立，$\sf n$维向量证明；

比如求 $f ( x ) = { \sqrt { m { \bigl ( } x + p { \bigr ) } } } + { \sqrt { n { \bigl ( } - x + q { \bigr ) } } }$ 最大值：

$$
$$\Rightarrow
\left\{\begin{array}{l}
a=\sqrt{m},b=\sqrt{n},c=\sqrt{x+p},d=\sqrt{-x+q} \\
 \\
\sqrt{m\left(x+p\right)}+\sqrt{n\left(-x+q\right)}\leq\sqrt{\left(\sqrt{m}^{2}+\sqrt{n}^{2}\right)}\sqrt{\left(\sqrt{x+p}^{2}+\sqrt{-x+q}^{2}\right)}=\sqrt{m+n}\sqrt{q+q} \\
 \\
\text{当且仅当}\frac{\sqrt{m}}{\sqrt{x+p}}=\frac{\sqrt{n}}{\sqrt{-x+q}}\text{时取等}; & 
\end{array}\right.$$
$$
