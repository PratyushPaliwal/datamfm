## 款冬花 KUANDONGHUA

【传统应用】用于咳嗽，化痰止咳，凡所有咳嗽，皆可使用。谓本品得紫菀良，二者相须而用，其效更佳。亦可与温肺之干姜及紫菀、五味子同用。若咳嗽偏热，可与知母、桑叶、贝母等同用。肺气虚弱、咳嗽不已，可与人参、白术、甘草等同用。若肺痈咯血，可与薏苡仁、桔梗等同用。用于喘证，可与麻黄、白果、桑皮等同用。

【成分与药理】花含生物碱（款冬花碱、克氏千里碱）、倍半萜、款冬花酮以及三萜、黄酮苷、挥发油、氨基酸、无机元素等。药理作用有镇咳、祛痰、平喘、呼吸兴奋、升压、强烈收缩血压、抑制血小板活化因子（PAF）等。

【现代临床研究与应用】（1）治疗哮喘：其醇浸膏每次限$5\mathrm { m l }$（相当生药 $6 \mathrm { g }$），每日3次。（2）治疗慢性支气管炎：款冬花和地龙制成注射剂，每次肌注 $2 \mathrm { m l }$ ，连续用药10天。款冬花煎剂小鼠灌服的 $\mathrm { L D } _ { 5 0 }$ 为 $1 2 4 \mathrm { g / k g }$ ；醇提取物腹腔注射$\mathrm { L D } _ { 5 0 }$为 $1 1 2 \mathrm { g / k g }$ ；醚提取物腹腔注射的$\mathrm { L D } _ { 5 0 }$为 $4 3 \mathrm { g / k g }$ 。款冬花有致癌活性。
