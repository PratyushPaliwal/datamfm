# # 理顺知识点

- 月经期子宫内膜剥脱后，雌激素能促进子宫内膜细胞增生，修复受损的子宫内膜

- 雌激素在月经周期的增生期起主要作用，促进子宫内膜细胞增生、增厚

- 孕激素在月经周期的分泌期起主要作用，主要是使子宫内膜分泌增加

- 雌激素加强子宫运动→子宫平滑肌细胞增生肥大、对缩宫素敏感性增高；孕激素减弱子宫运动

- 排卵期宫颈粘液增多、稀薄，排卵期宫颈口松弛→有利于精子和卵子相遇

- 孕激素提示机体处于怀孕状态→阻止精子通过→宫颈粘液减少、粘稠

- 阴道上皮细胞属于鳞状上皮细胞→会角化

- 女性生殖器官发育，第二性征的形成都与雄激素、雌激素有关，而与孕激素无关

- 动脉粥样硬化好发于中老年男性→雌激素有利于脂代谢→血 LDL↓，HDL↑
