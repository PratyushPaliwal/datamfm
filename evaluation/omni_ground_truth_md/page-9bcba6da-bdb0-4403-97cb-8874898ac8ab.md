# ### 1.1.6【添项配凑类型】

# #### ▲适用题型：

添项配凑后可以构造利用不等式的题型；

# #### ▲方法原理：

常用添项配凑方法：

（1）单变量观察分母：$\left\{
\begin{array}{l}
x + \frac{k}{x-a} = (x-a) + \frac{k}{(x-a)} + a \\
x + \frac{k}{mx-a} = \frac{(mx-a)}{m} + \frac{k}{(mx-a)} + \frac{a}{m}
\end{array}
\right.$

（2）双变量观察分母：$\begin{cases}
2a+3b+\frac{1}{a+b}+\frac{1}{a+2b}=a+b+\frac{1}{a+b}+a+2b+\frac{1}{a+2b} \\
 \\
\frac{a}{b}+\frac{b}{a+b}=\frac{a+b-b}{b}+\frac{b}{a+b}=\frac{a+b}{b}+\frac{b}{a+b}-1 \\
 \\
\frac{3b+2a}{b}+\frac{b}{a+b}=\frac{b+2b+2a}{b}+\frac{b}{a+b}=1+\frac{2\left(a+b\right)}{b}+\frac{b}{a+b} \\
 \\
a+\frac{1}{a-b}+\frac{1}{a+b}=a-b+\frac{1}{a-b}+a+b+\frac{1}{a+b} & 
\end{cases}$

（3）平方添项：${ \frac { a - 3 b } { a ^ { 2 } + 9 b ^ { 2 } } } = { \frac { a - 3 b } { \left( a - 3 b \right) ^ { 2 } + 6 a b } } = { \frac { 1 } { a - 3 b + { \frac { 6 a b } { a - 3 b } } } }$

（4）同乘易化简的结构：$\left\{ \begin{array} { l } { a + b = 1 + \frac { 1 } { a } + \frac { 1 } { b } } \\ {\Rightarrow \big ( a + b \big ) ^ { 2 } = \left( 1 + \frac { 1 } { a } + \frac { 1 } { b } \right)  ( a + b ) =  ( a + b) + 2 + \frac { b } { a } + \frac { a } { b } } \end{array} \right.$

以上只是列举一部分配凑变形的方式，核心在于朝基本不等式方向构造，多观察题干条件，合理配凑，当然还得多练习.
