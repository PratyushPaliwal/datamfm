# 一轮复习 ：中考化学基本理论有关化学式的计算

【例1】氮肥中的含氮量可以使用这样的方法鉴定 ：将氮肥（ 有效成分为铵盐 ）配制成溶液，向溶液中滴加过量的 NaOH溶液，可以发生如下反应 ( 以 $ \mathbf{N H}_{4} \mathbf{C} \mathbf{l} $为例 ) ： $ \mathbf{N H}_{4} \mathbf{C} \mathbf{I}+\mathbf{N a O H}=\mathbf{N a C I}+\mathbf{H}_{2} \mathbf{O}+\mathbf{N H}_{3} \uparrow $再将氨气通入稀硫酸中，生成硫酸铵.最终测定消耗的硫酸质量即可求出氮肥中氮元素的质量。现从 100kg某氮肥（ 主要成分为硝酸铵，杂质不含N元素 ）中取 6g配制成溶液，加入足量的 NaOH溶液，用稀硫酸吸收产生的氨气，后测得消耗质量分数为 4.9%的硫酸 50g ，则原氮肥中氮元素的总质量为 (    )
A. 11.67kg
B. 11.88kg
C. 23.33kg
D. 23.76kg

C. 45%
D. 29%

A. 71 : 75
B.71 : 79
