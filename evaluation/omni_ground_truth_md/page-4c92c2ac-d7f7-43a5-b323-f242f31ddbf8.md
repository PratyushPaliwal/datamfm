# # 相关OPC算法

# ## 算法简介及其优缺点

# - 基于EPE的OPC相关算法

基于最小边缘放置误差（EPE）的优化算法有不少，掩模误差增强因子（MEEF）矩阵广泛应用于此。

优点：模型经典，应用广泛

缺点：随CD发展计算时间大大增长

# - PW-OPC

工艺窗口（PW）OPC，同时能够考虑工艺窗口和EPE变化，应对能量和聚焦的不稳定性，将图形尺寸偏差可以控制在2nm以内

优点：能够发现图形DOF的不足，解决绝大多数坏点问题

缺点：修出来的版图容易受版图规则限制，导致可制造性降低
