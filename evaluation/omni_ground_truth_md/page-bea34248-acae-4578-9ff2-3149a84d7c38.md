设$f ( x ) = \left\{ \begin{array} { l l } {  \operatorname* { l i m } _ { n \to \infty } \frac { 1 } { n } \bigg ( 1 + \cos \frac { x } { n } + \cos \frac { 2 x } { n } + \dots + \cos \frac { n - 1 } { n } x \bigg ) , } & { x > 0 , } \\ { a , } & { x = 0 , } \\ { f ( - x ) , } & { x < 0 } \end{array} \right.$ 连续，则 $a =$____.

解 应填1.

当 $x > 0$ 时，$f\left(x\right)=\lim\limits_{n\to\infty}\frac{1}{n}\sum\limits_{i=0}^{n-1}\cos\frac{x}{n}i=\lim\limits_{n\to\infty}\frac{1}{x}\sum\limits_{i=0}^{n-1}\cos\frac{x}{n}i\bullet\frac{x}{n}$

$$
$$=\frac{1}{x}\int_{0}^{x}\cos t\mathrm{d}t=\frac{1}{x}\sin t|_{0}^{x}=\frac{\sin x}{x};$$
$$

当x<0时，$f(x)=f(-x)=\frac{\sin(-x)}{-x}=\frac{\sin x}{x}.$

综上所述， $f\left(x\right)=
\begin{cases}
\frac{\sin x}{x}, & x\neq0, \\
a, & x=0. &
