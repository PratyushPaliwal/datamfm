# # 案例38.托架、集装箱号等申报不实案例

# ## 一、基本案情

2019年×月×日，司机张某某驾驶粤ZXX港车，捆绑××号载货清单，从皇岗口岸承运货物出境。经查，该车集装箱申报重量5500KG，实际重量4700KG。广东鸿兴运输有限公司在舱单确报环节传输集装箱重量的电子数据不准确，被查获。

# ## 二、案件办理分析

# ### （一）定性判断。

首先，《海关法》第十四条规定了运输工具负责人的申报义务，因此，未按规定时限传输电子数据、传输的舱单电子数据不准确，其行为已构成违法。其次，在现有证据材料的基础上，无法充分证明涉案公司的申报不实存在故意违反海关法律法规规定的主观意图，其行为不构成走私行为，应当定性为违反海关监管规定的行为（违规）。

《海关法》第十四条规定“进出境运输工具到达或者驶离设立海关的地点时，运输工具负责人应当向海关如实申报，交验单证，并接受海关监管和检查······”、《中华人民共和国海关行政处罚实施条例》第十二条规定“违反海关法及其他有关法律、行政法规和规章但不构成走私行为的，是违反海关监管规定的行为”。

# ### （二）立案判断。

该案舱单电子数据传输不准确属于行为罚，根据《海关法》第八十六条第（十三）项规定，应当立案处罚。

## 第六章 违反海关法及海关行政处罚实施条例案例

立案依据：根据《海关法》第八十六条第（十三）项规定“违反本法规定有下列行为之一的，可以处以罚款，有违法所得的，没收违法所得：（十三）有违反海关监管规定的其他行为的”。

# ### （三）处罚分析。

当事人一年内在深圳关区无同类违规记录，根据《中华人民共和国海关行政处罚实施条例》第五十三条之规定，对该申报不实行为判定为一般情节（无从轻、减轻、从重情节）。运输工具负责人有如实向海关申报、交验单证的义务，该案中运输工具负责人（舱单传输确报环节）因传输舱单电子数据不准确导致单货不符，实际处罚根据《中华人民共和国海关行政处罚实施条例》第二十二条（四）项规定确定。

《中华人民共和国海关行政处罚实施条例》第五十三条（二）项之规定“因违反海关监管规定被海关行政处罚后在1年内又实施同一违反海关监管规定的行为的”、《中华人民共和国海关行政处罚实施条例》第二十二条（四）项规定“有下列行为之一的，予以警告，可以处5万元以下罚款，有违法所得的，没收违法所得：（四）未按照规定期限向海关传输舱单等电子数据、传输的电子数据不准确或者未按照规定期限保存相关电子数据，影响海关监管的；”《海关办理行政处罚简易程序案件、简单案件操作规程（试行）》（署缉发〔2017〕205号）第一次警告，第二次及之后每次予以警告并处以1000元罚款。
