$$
$$4\left( {x - 1}\right)  + 5\left( {y - 2}\right)  + 3\left( {z - 3}\right)  = 0.$$

化简之后变成

$$
$${4x} + {5y} + {3z} = {23}.$$

例题2 你的表哥说地板应该没问题，准是你的法向量歪了．于是哥儿俩走出厨房，来到格局呈三角形的客厅，一起量得三个角落的位置分别是 $P ( 1 , 2, 3 )$、$Q(5,0,1)$及 $R ( 0 , 4 , 1 )$ ，试利用这三个点，重新计算包含你家地板的平面方程式.

解：题目给了我们三个点，但是没有法向量 $\pmb { n }$ ，所以当务之急是要先求出$\pmb { n }$ 来．如图5.18所示，由于三个点全在地板平面上，因此向量$\overrightarrow {PQ}$跟$\overrightarrow {PR}$必然都躺在地板上，它们的叉积就会跟地板垂直，正好可当做一个法向量．于是，

$$
$$\overrightarrow{PQ} = \left( {5 - 1,0 - 2,1 - 3}\right)  = \left( {4, - 2, - 2}\right) ,$$
$$

$$
$$\overrightarrow{PR} = \left( {0 - 1,4 - 2,1 - 3}\right)  = \left( {-1,2, - 2}\right) ,$$
$$

$$
$${ \overrightarrow { P Q } \times \overrightarrow { P R } = \left( { 8 , 1 0 , 6 } \right) . }$$
$$

图5.18 法向量就是向量$\overrightarrow {PQ}$跟$\overrightarrow {PR}$的叉积
