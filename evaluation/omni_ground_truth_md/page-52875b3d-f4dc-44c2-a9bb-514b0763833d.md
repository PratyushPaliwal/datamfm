# #### 1.2.1.4【待定系数类型】

# # $\blacktriangle$适用题型：

知道可以配凑出${ ( } { \frac { m } { a } } + { \frac { n } { b } } { ) } {  ( } p a + q b {  ) }$结构，但是不能直接观察出系数的不等式题型；如：已知正实数$x$、$y$满足${ \frac { 1 } { x + 3 y } } + { \frac { 1 } { 2 x + y } } = 1$，则$x+y$的最小值为____·

# # $\blacktriangle$方法原理：

1. 如果可以直接看出或者简单尝试可知系数关系的，则直接处理

比如：$x>0,y>0,x+2y=1,$则$\frac{1}{4x+3}+\frac{1}{2y+1}$的最小值为_____.

变形：$\begin{cases}
x+2y=\frac{1}{4}\left(4x+3\right)+\left(2y+1\right)-\frac{7}{4}=1\Longrightarrow\left(4x+3\right)+4\left(2y+5\right)=11 \\
\frac{1}{4x+3}+\frac{1}{2y+5}=\frac{1}{11}\left(\frac{1}{4x+3}+\frac{1}{2y+5}\right)\left[\left(4x+3\right)+4\left(2y+5\right)\right] \\
\text{此时建议结合换元处理结构更简单}\begin{cases}4x+3=m\\
2y+1=n
\end{cases}
\end{cases}$

2. 不易看出系数关系的,则待定系数处理:

比如: $x > 0 , y > 0 , 3 x + y = 1$ ,则 ${ \frac { 1 } { 5 x + y } } + { \frac { 1 } { 5 x + 3 y } }$ 的最小值为____.

变形: $\begin{cases}
 3x+y=m\left(5x+y\right)+n\left(5x+3y\right)\Rightarrow
\begin{cases}
5m+5n=3 \\
m+3n=1 
\end{cases}\Rightarrow
\begin{cases}
m=\frac{2}{5} \\
n=\frac{1}{5} 
\end{cases} \\
\Rightarrow\frac{1}{5x+y}+\frac{1}{5x+3y}=\left(\frac{1}{5x+y}+\frac{1}{5x+3y}\right)\left[\frac{2}{5}\left(5x+y\right)+\frac{1}{5}\left(5x+3y\right)\right] \\
 \text{此时建议结合换元处理结构更简单}
\begin{cases}
5x+y=m \\
5x+3y=n
\end{cases}
\end{cases}$
