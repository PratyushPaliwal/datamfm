# ## 3.3.2.3【轴心加单调类型】

$\blacktriangle$适用题型:已知对称轴和对称中心以及单调性，求$\omega$范围的题目，并且（不论开闭）区间不含0，比如$[ \frac { \pi } { 4 } , \frac { \pi } { 3 } ]$；

# $\blacktriangle$方法原理：

$y = \sin \left( w x + \varphi \right)$对称轴$x = p$，对称中心$( t , 0)$，在$ ( a , b )$单调，则$w$范围：

1. 法一：卡形式卡范围再讨论

（1）卡形式：轴$x = p$，心$( t , 0) \Rightarrow \left\{ \begin{array} { l l } {  \left| p - t \right| = \frac { \left( 2 n - 1 \right) T } { 4 } = \frac { \left( 2 n - 1 \right) } { 4 } \frac { 2 \pi } { w } \Bigg [ \text{常写成}\frac { (2n+1)T } { 4 } , n\in N}\Bigg] \\ {  \Rightarrow w = \frac { \left( 2 n - 1 \right) \pi } { 2 \left| p - t \right| } \Big ( n \in N ^ { * } \Big ) } \end{array} \right.$

（2）卡范围：$| b - a | \leq \frac { T } { 2 } \Rightarrow \left| b - a \right| \leq \frac { 1 } { 2 } \cdot \frac { 2 \pi } { w } \Rightarrow w \leq \frac { \pi } { \left| b - a \right| }$

（3）讨论验证：$\left\{ \begin{array} { l } {  { w = \frac { \left( 2 n - 1 \right) \pi } { 2 \left| p - t \right| } } } \\ {  { w \leq \frac { \pi } { \left| b - a \right| } } } \end{array} \right. \Rightarrow\text{逐个讨论}w\left\{\begin{array}{l}  \langle 1\rangle w\text{取一个符合的值}\\\langle 2 \rangle\begin{aligned}\text{对称中心}(t,0)\Rightarrow{tw+\varphi=k\pi} \\\varphi\text{的范围}（k\in Z）\end{aligned}\Big\}\Rightarrow\varphi \\\langle 3 \rangle\text{验证单调性是否符合题意}\\\langle 4 \rangle\text{按以上步骤循环讨论验证直至求出答案}
\end{array}\right.$

注意：以上结论无需记忆，重在理解；
