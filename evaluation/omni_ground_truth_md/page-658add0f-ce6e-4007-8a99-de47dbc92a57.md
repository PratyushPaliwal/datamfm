# # 二 祈使句

# ## 1 肯定形式

- 动词原形 + 其他。

Please come here. 请过来。

- Be + 表语 + 其他。

Be careful. 小心点！

- Let + 宾语 + 动词原形 + 其他。

Let him go there alone. 让他一个人去。

# ## 2 否定形式

- Don't + 动词原形 + 其他。

Don’t smoke here！

不要在这抽烟！

- Don't be + 表语 + 其他。

Don't be late again. 不要又迟到了。

- Let + 宾语 + not + 动词原形 + 其他；Don't let + 宾语 + 动词原形 + 其他。

Let him not go out.

= Don't let him go out.

不要让他出去。

- No + 名词/动名词。

No parking. 不准停车。

- Never + 动词。

Never mind. 不要紧。
