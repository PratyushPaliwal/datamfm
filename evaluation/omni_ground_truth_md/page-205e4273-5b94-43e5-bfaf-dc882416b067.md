# # 气和隔立消——小儿打隔的复方调理法

打隔就是膈肌癌挛。嬰儿与成年人都有膈肌，但功能略有不同，嬰儿以腹式呼吸为主，膈肌还是呼吸肌的一部分。当婴儿吃奶过快或吸入冷空气时，就会使膈肌突然收缩，引起迅速吸气并发出“隔”的声音。这种情况一般都能自行缓解，但如果持续性发作或间歇性发作，就需要调理了。

# 【原因】

胃寒、胃热、食滞、气郁和正气亏虚等造成膈肌痉挛。
