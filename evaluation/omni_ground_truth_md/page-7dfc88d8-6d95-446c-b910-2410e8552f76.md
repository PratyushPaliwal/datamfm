## 南方区域电力市场现货电能量交易实施细则（试行V3.0）

$$
$$\min \left\{  \begin{array}{l} \mathop{\sum }\limits_{{i = 1}}^{N}\mathop{\sum }\limits_{{t = {t}_{start}}}^{T}\left\lbrack  {{C}_{i,t}\left( {P}_{i,t}\right) {d}_{t}}\right\rbrack   + \mathop{\sum }\limits_{{i = 1}}^{n}\mathop{\sum }\limits_{{t = {t}_{\text {start}}}}^{T}{P}_{gwf}\left\lbrack  {{P}_{L,i,t}{d}_{t}}\right\rbrack   + \mathop{\sum }\limits_{{l = 1}}^{{NL}}\mathop{\sum }\limits_{{t = {t}_{\text {start}}}}^{T}{M}_{1}\left\lbrack  {S{L}_{t}^{ + } + S{L}_{t}^{ - }}\right\rbrack   + \mathop{\sum }\limits_{{s = 1}}^{{NS}}\mathop{\sum }\limits_{{t = {t}_{\text {start}}}}^{T}{M}_{1}\left\lbrack  {S{L}_{s}^{ + }}+\right. \\  \left. {+\mathop{\sum }\limits_{{r = 1}}^{{NR}}\mathop{\sum }\limits_{{t = 1}}^{T}{M}_{2}{P}_{r,t}^{d} + \mathop{\sum }\limits_{{i = 1}}^{{NH}}\mathop{\sum }\limits_{{t = 1}}^{T}{M}_{3}{P}_{i,t}^{d} + \mathop{\sum }\limits_{{k \in  {\Phi }_{n}}}\mathop{\sum }\limits_{{t = 1}}^{T}{M}_{4}\left\lbrack  {S{L}_{k}^{ - }}\right\rbrack   + \mathop{\sum }\limits_{{es = 1}}^{{ES}}\mathop{\sum }\limits_{{t = 1}}^{T}\left\lbrack  {{\lambda }_{es}^{dis}{P}_{es,t}^{dis} + {\lambda }_{es}^{ch}{P}_{es,t}^{ch}}\right\rbrack  }\right.  \end{array}\right.$$
$$

其中：

$N$表示南方区域发电机组的总台数，不包含储能交易单元；

$T$ 表示每次的总时段数，即24； $t _ {\text { start} }$ 表示滚动更新开始的时间，因此考虑时段的数量为 $T - t _ { \text { start} }$；

$d _ { t }$代表时段长度，取5分钟；

$P _ { i , t }$表示机组$i$在时段$t$的出力；

机组运行费用$C _ { i , t } \left( P _ { i , t } \right)$是与机组申报的各段出力区间和对应能量价格有关的多段线性函数；

$n$表示南方区域跨省输电通道的数量；$P _ { L , i , t }$为跨省送电成分$i$在时段$\mathrm { t }$的输电功率；$P _ { g w f }$为跨省输电的过网费；

$M _ { 1 }$为网络潮流约束松弛罚因子；

$S L _ { l } ^ { + }$、$S L _ { l } ^ { - }$分别为线路1的正、反向潮流松弛变量；$NS$为线路总数；

$S L _ { s } ^ { + }$、$S L _ { s } ^ { - }$分别为断面$s$的正、反向潮流松弛变量；$NL$为断面总数；

$M _ { 2 }$为新能源交易单元弃电罚因子，$NR$为有弃电功率新能源场站总数；

$P _ { r , t } ^ { d }$为新能源交易单元$r$在时段$t$的弃电功率；

$M _ { 3 }$为水电厂弃水功率罚因子；

$P _ { i , t } ^ { d }$为水电厂$i$的弃水功率，根据其弃水流量计算；
