# ### A.4.2 制冷季节耗电量

制冷季节耗电量CSTE按公式 (A.6) 计算。

$$
$$\operatorname{CSTE} = \mathop{\sum }\limits_{{j = 1}}^{n}\left\lbrack  \frac{{Q}_{\mathrm{c}}\left( {t}_{j}\right) }{{\mathrm{{COP}}}_{\text{bin }}\left( {t}_{j}\right) }\right\rbrack   \times  {n}_{j} \cdots \tag{A.6}$$
$$

式中：

$\mathrm{COP_{bin}}(t_j)$——各工作温度下的制冷性能系数；

$\mathrm{COP_{bin}}(t_j)$——通过测试并按公式(A.7)～公式(A.11)计算获得。

a）当室外温度在$43\degree \mathrm{C} $～$35\degree \mathrm{C} $（A点）之间时：

$$
$${\mathrm{{COP}}}_{\mathrm{{bin}}}\left( {t}_{j}\right)  = {\mathrm{{COP}}}_{\mathrm{{bin}}}\left( {t}_{\mathrm{A}}\right)  + \frac{{\mathrm{{COP}}}_{\mathrm{{bin}}}\left( {43}\right)  - {\mathrm{{COP}}}_{\mathrm{{bin}}}\left( {t}_{\mathrm{A}}\right) }{{43} - {t}_{\mathrm{A}}} \times  \left( {{t}_{j} - {t}_{\mathrm{A}}}\right) ,{t}_{\mathrm{A}} \leqslant  {t}_{j} \cdots\tag{A.7}$$
$$

b）当室外温度在$35\degree \mathrm{C} $（A点）～$32.3\degree \mathrm{C} $（B点）之间时：

$$
$${\mathrm{{COP}}}_{\mathrm{{bin}}}\left( {t}_{j}\right)  = {\mathrm{{COP}}}_{\mathrm{{bin}}}\left( {t}_{\mathrm{B}}\right)  + \frac{{\mathrm{{COP}}}_{\mathrm{{bin}}}\left( {t}_{\mathrm{A}}\right)  - {\mathrm{{COP}}}_{\mathrm{{bin}}}\left( {t}_{\mathrm{B}}\right) }{{t}_{\mathrm{A}} - {t}_{\mathrm{B}}} \times  \left( {{t}_{j} - {t}_{\mathrm{B}}}\right) ,{t}_{\mathrm{B}} \leqslant  {t}_{j} \leqslant  {t}_{\mathrm{A}}\cdots \tag{A. 8}$$
$$

c）当室外温度在$32.3\degree \mathrm{C} $（B点）～$29.5\degree \mathrm{C} $(C点)之间时：

$$
$$\mathrm { C O P } _ { \mathrm { bin } } ( t _ { j } ) = \mathrm { C O P } _ { \mathrm { bin } } ( t _ { \mathrm { c } } ) + \frac { \mathrm { C O P } _ { \mathrm { bin } } ( t _ { \mathrm { b } } ) - \mathrm { C O P } _ { \mathrm { bin } } ( t _ { \mathrm { c } } ) } { t _ { \mathrm { B } } - t _ { \mathrm { c } } } \times ( t _ { j } - t _ { \mathrm { c } } ) , t _ { \mathrm { c } } \leqslant t _ { j } \leqslant t _ { \mathrm { B } } \cdots \tag{A.9}$$
$$

d）当室外温度在$29.5\degree \mathrm{C} $（C点）～$26.8\degree \mathrm{C} $（D点）之间时：

$$
$${\mathrm{{COP}}}_{\text{bin }}\left( {t}_{j}\right)  = {\mathrm{{COP}}}_{\text{bin }}\left( {t}_{\mathrm{D}}\right)  + \frac{{\mathrm{{COP}}}_{\text{bin }}\left( {t}_{\mathrm{C}}\right)  - {\mathrm{{COP}}}_{\text{bin }}\left( {t}_{\mathrm{D}}\right) }{{t}_{\mathrm{C}} - {t}_{\mathrm{D}}} \times  \left( {{t}_{j} - {t}_{\mathrm{D}}}\right) ,{t}_{\mathrm{D}} \leqslant  {t}_{j} \leqslant  {t}_{\mathrm{C}}\cdots \tag{A.10}$$
$$

e）当室外温度$26.8\degree \mathrm{C} $（D点）～$24\degree \mathrm{C} $（0负荷点）之间时：

$$
$$\mathrm{COP}_{\mathrm{bin}}(t_{j})=\mathrm{COP}_{\mathrm{bin}}(t_{\mathrm{D}})-0.0289\times\mathrm{COP}_{\mathrm{bin}}(t_{\mathrm{D}})\times(t_{j}-t_{\mathrm{D}}),24\leqslant t_{j}\leqslant t_{\mathrm{D}}\cdots\tag{A.11}$$
$$
