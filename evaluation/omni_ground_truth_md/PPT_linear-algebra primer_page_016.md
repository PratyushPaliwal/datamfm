# Determinant of a Matrix

Used for inversion

If det $(A) = 0$, then A has no inverse

$$
$$
A = { \left[ \begin{array} { l l } { a } & { b } \\ { c } & { d } \end{array} \right] }~~\det( A ) = a d - b c
$$
$$

$$
$$
A ^ { - 1 } = { \frac { 1 } { a d - b c } } { \left[ \begin{array} { l l } { d } & {-b } \\ { - c } & { a } \end{array} \right] }
$$
