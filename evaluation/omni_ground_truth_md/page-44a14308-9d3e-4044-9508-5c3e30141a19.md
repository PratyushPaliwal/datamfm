# ### 2.11.3【能成立求参】

▲适用题型：

一元二次不等式能成立（存在性或有解问题）求参数问题；

▲方法原理：

1. 参数一次时考虑分参处理；

2. 二次项系数未知时可考虑讨论或者分参；

3. 还可考虑转化为恒成立问题，利用命题的否定与原命题所计算的答案互补为R来处理；

比如：若关于 $x$ 的不等式 $a x ^ { 2 } + 3 x + 1 < 0$ 有实数解，则 $a$ 的取值范围是____.

解：

$$
$$\left\{  \begin{array}{l} \exists x \in  R,a{x}^{2} + {3x} + 1 < 0\text{成立} \\   \Rightarrow  \text{否定: }\forall x \in  R,a{x}^{2} + {3x} + 1 \geq  0\text{成立} \Rightarrow  \left\{  \begin{array}{l} a = 0,{3x} + 1 \geq  0\text{不恒成立, 舍} \\  a > 0,\Delta  \leq  0 \Rightarrow  a \geq  \frac{9}{4} \end{array} \right.\end{array}\right\} \Rightarrow a < \frac{9}{4}$$
$$

例题1．（全国 • 高三专题练习）若关于 $x$ 的不等式$a x ^ { 2 } + 2 x + 1 < 0$有实数解，则 $a$ 的取值范围是____.

例题2．（全国 • 高三专题练习）不等式 $x ^ { 2 } + a x - 2 > 0$ 在区间$[1,5]$上有解，则 $ { a }$ 的取值范围是____.

例题3．（全国 • 高一课时练习）若不等式 $a x ^ { 2 } + x + 1 > 0$ 在 $x \in \left[ 1 , 2 \right]$ 时有解，则实数${ a }$的取值范围为____.

例题4．（江苏 • 高一单元测试）若关于 $x$ 的不等式 $x ^ { 2 } + ( m - 1 ) x + 1 \leq 0$ 在区间$(0,2]$上有解，则实数$m$的取值范围是 ____.
