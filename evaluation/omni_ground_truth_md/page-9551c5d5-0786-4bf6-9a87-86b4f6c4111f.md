# # （二）团体智力测验

# ## 1.陆军测验

最初用来招募和选拔士兵。

# ### （1）陆军甲种

由8个分测验组成，以英文呈现，属于文字测验；第一个团体施测的智力测验。

# ### （2）陆军乙种

由7个分测验组成，属于非文字测验；适用于母语非英语和文盲;与甲种测验的相关达到0.8。

# ## 2.瑞文推理测验

基于智力二因素中G因素理论的非文字型的图形测验，可个别施测，结果先转化为百分等级，再转化为IQ分数，测量一般智力。

瑞文将智力的G因素分为两种独立的能力：

（1）再生性能力：经过教育后达到的水平，适合词汇测验

（2）推断性能力：不受教育影响的理性判断能力，适合非语言的图形推理测验。

瑞文测验分为四个水平：

（1）瑞文标准推理测验（SPM）

1938年编制，由5个单元构成，共60个项目;适用于8岁以上的被试；属于中等水平的瑞文测验。

（2）瑞文彩图推理测验（CPM）

1947年编制，有3个系列，共36个项目；适用于5.5~11岁的儿童和智力落后的成人；属于最低水平的瑞文测验。

（3）瑞文高级推理测验（APM）

1941年编制，适用于高智力成人；是最高水平的瑞文推理测验。
