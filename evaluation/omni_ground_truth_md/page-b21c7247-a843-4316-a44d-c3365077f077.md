# ### 2.3.11【构造函数法之双根式】

# $\blacktriangle$适用题型：

常用于单调性未知的根式型 $f \left( x \right) = \sqrt { a \left( x - b \right) } + \sqrt { c \left( d - x \right) } \left( a c > 0 \right)$ 求值域；

# #### $\blacktriangle$方法原理：构造一个满足三个条件的新函数：

①与原函数定义域一致 ②在定义域上单调 ③且与原函数平方和为定常数，即可利用所构造函数的值域求出 $f ( x )$ 的值域.

怎么构造？

比如：求 $f \left( x \right) = { \sqrt { 3 x - 6 } } + { \sqrt { 8 - 2 x } } , x \in \left[ 2 , 4 \right]$ 值域

(1)由于$f \left( x \right) = \sqrt { 3 \left( x - 2 \right) } + \sqrt { 2 \left( 4 - x \right) }$前后平方后出现$3 ( x - 2 )$和$2 ( 4 - x )$，为了消去$ { x } $我们需要新函数含有$- 3 x$和$ { 2x }$；

(2)为了保持与原函数定义域一致，我们再进行添项变成$3 ( 4 - x )$和$2 ( x - 2 )$；

(3)考虑新函数单调性和与原函数平方和为定值，所以构造$- { \sqrt { 3 { \bigl ( } 4 - x { \bigr ) } } } + { \sqrt { 2 { \bigl ( } x - 2 { \bigr ) } } }$，从而得到$\begin{cases}
h(x)=-\sqrt{3\left(4-x\right)}+\sqrt{2\left(x-2\right)}\uparrow \\
-\sqrt{6}=h(2)\leq h(x)\leq h(4)=2 & 
\end{cases}$，且$f ^ { 2 } ( x ) + h ^ { 2 } ( x ) = 1 0$，最后

$$
$$\left\{ \begin{array} { l l } { h ^ { 2 } ( x ) \in \left[ 0 , 6 \right] } \\ { f ^ { 2 } ( x ) = 1 0 - h ^ { 2 } ( x ) \in \left[ 4 , 1 0 \right] } \end{array} \right. \Rightarrow f ( x ) \in \left[ 2 , \sqrt { 1 0 } \right] ;$$
$$

总结构造新函数技巧（快速构造）：

$$
$$\left\{
\begin{array}{l}
f(x) = \sqrt{a(x-b)} + \sqrt{c(d-x)} \Rightarrow h(x) = -\sqrt{a(d-x)} + \sqrt{c(x-b)} \\
{(x-b)和(d-x)\text交换位置,} \sqrt{a(d-x)} \text{前再添负号}
\end{array}
\right.$$
$$
