# ## （二）$ { n }$母线系统

$n$ 母线功率方程是这类方程的一般形式。由式并用结点注入功率代替结点电流后可得

$$
$$\frac{{P}_{i} - j{Q}_{i}}{ \overset{*}{\dot{U_i}} } = \mathop{\sum}\limits_{{j = 1}}^{n}{Y}_{ij}{U}_{j}\qquad i = 1,2,\ldots ,n$$
$$

式中 $P _ { i } $ 、$Q _ { i }$分别为结点 $i$ 的注入有功功率和注入无功功率，因而有

$$
$${P}_{i} - j{Q}_{i} = \overset{*}{\dot{U_i}}\mathop{\sum }\limits_{{j = 1}}^{n}{Y}_{ij}{U}_{j}\qquad i = 1,2,\ldots ,n$$
$$

使 $\dot { U } _ { i } = U _ { i } e ^ { j  {\delta } _ { i } } \mathrm { ~ } , Y _ { i j } = G _ { i j } + \mathrm { j } B _ { i j }$，则有

$$
$${P}_{i} - \mathrm{j}{Q}_{i} = {U}_{i}{e}^{-\mathrm{j}{\delta }_{i}}\mathop{\sum }\limits_{{j = 1}}^{n}\left( {{G}_{ij} + \mathrm{j}{B}_{ij}}\right) {U}_{j}{e}^{j{\delta }_{j}}= {U}_{i}\mathop{\sum }\limits_{{j = 1}}^{n}\left( {{G}_{ij} + \mathrm{j}{B}_{ij}}\right) {U}_{j}{e}^{-{j}({\delta }_{i} - {\delta }_{j})}$$
$$

$$
$$i = 1,2,\ldots ,n$$
$$

由于

$$
$$e ^ { - j( \delta_ { i } - \delta_ { j } ) } = \cos ( \delta_ { i } - \delta_ { j } )-\mathrm j\sin ( \delta_ { i } - \delta_ { j } )$$
$$

故有

$$
$${P}_{i} - j{Q}_{i} = {U}_{i}\mathop{\sum }\limits_{{j = 1}}^{n}{U}_{j}\left\{  {{G}_{ij}\cos \left( {{\delta }_{i} - {\delta }_{j}}\right)  + {B}_{ij}\sin \left( {{\delta }_{i} - {\delta }_{j}}\right) }\right. $$
$$

$$
$$+j[\  B _ { i j } \mathrm { c o s } ( {{\delta }_{i} - {\delta }_{j}}) - G _ { i j } \mathrm { s i n } ( {{\delta }_{i} - {\delta }_{j}})] \}$$

$$
