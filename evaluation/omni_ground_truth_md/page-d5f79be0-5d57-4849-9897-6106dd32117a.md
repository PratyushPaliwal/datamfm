# # 3．胡克定律

(1)内容：在弹性限度内，弹簧弹力 $F$ 的大小与弹簧的形变量 $x$ 成正比(定律的成立是有条件的，也就是弹簧发生“弹性形变”，即必须在弹性限度内.

(2)公式： $F = k x$ ，式中 $k$ 为弹簧的劲度系数，单位是牛顿每米，符号是 $\textnormal { N } / \textnormal { m }$.

(3)图象：根据胡克定律，弹力与弹簧形变量的关系可用 $F - x$ 图象表示，如右图.这是一条通过原点的倾斜直线，其斜率 $\tan \alpha = { \frac { F } { x } } = k$.

(4)利用 $F - x$ 图象，很容易得到胡克定律的另一种表达式 $\Delta F = k \cdot \Delta x$ ， $\Delta x$ 是弹簧长度的变化量， $\Delta F$ 是弹力的变化量.

# 说明：

$\textcircled{1} F { = } k x$ 中的 $x$ 是弹簧的形变最，是弹簧伸长(或缩短)的长度，而不是弹簧的总长度.

$\textcircled { 2 } F { = } k x$ 中的 $k$ 是弹簧的劲度系数，它反映了弹簧的“软”“硬”程度，大小仅由弹簧本身的性质决定.

$\textcircled{3}$ 在应用 $F { = } k x$ 时，要把各物理量的单位统一到国际单位制中.

$\textcircled{4}$ 胡克定律的表达式 $ { F } = { k } \cdot  { x }$ ，仅表示弹簧的弹力与弹簧形变量之间的大小关系，若同时要表示弹力方向与弹簧形变方向相反，则表达式可写成 $F = - k \cdot x$ .
